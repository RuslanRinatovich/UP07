## 📌 Модуль 1. Установка и настройка SQL-сервера с автоматическим созданием пользователей и баз данных

### 🎯 Задача
Выберите СУБД и среду для управления инфраструктурой.  
Установите ядро выбранной СУБД и среду для управления инфраструктурой SQL (на виртуальную машину или представленный компьютер).

### ⚙️ Требования к установке
- Имя сервера: `Server_номер_рабочего_места` (пример: `Server_05`)
- Режим аутентификации: **смешанный**
- Пользователь `sa`: включить или создать  
- Пароль `sa`: `De_номер_рабочего_места` (пример: `De_05`)

### 📝 Необходимо разработать скрипт, который:

1. Создаёт пользователей `user1`, `user2`, …, `user10`
   - Пароль генерируется случайно
   - Длина пароля: **5 символов** (буквы + цифры)

2. Создаёт базы данных `BD1`, `BD2`, …, `BD10`

3. Настраивает права доступа:
   - `user1` → только `BD1`
   - `user2` → только `BD2`
   - и так далее

4. Создаёт базу данных `BD` и таблицу `Users` для хранения:
   - логинов
   - паролей

5. Заполняет таблицу `Users` данными о созданных пользователях и их паролях

## 🔐 Модуль 2. Шифрование паролей

### 🎯 Задача
Хранение паролей в зашифрованном виде — важное требование безопасности доступа к серверу.

### 📝 Необходимо разработать скрипт, который:

1. **Зашифрует** все пароли в таблице `Users`

2. **Расшифрует** пароли для отображения (без потери доступа к аккаунтам)

> ⚠️ Внимание: скрипт должен позволить безопасно отобразить данные из таблицы `Users` с расшифрованными паролями
