## 📌 Задача

Необходимо доработать существующую базу данных на основе предоставленного макета.

---

## 📝 Порядок выполнения

### 1️⃣ Изучите макет БД
Ознакомьтесь с макетом базы данных, представленным на рисунке ниже:

![Схема базы данных hotel](images/hotel%20-%20public.png)

### 2️⃣ Доработайте структуру своей БД
Внесите необходимые изменения в структуру существующей базы данных в соответствии с макетом.

### 3️⃣ Заполните таблицы тестовыми данными
Добавьте начальные тестовые данные в каждую таблицу (не менее **3 записей** в каждую таблицу).

### 4️⃣ Создайте словарь данных
Разработайте словарь данных для вашей базы данных.

> 📎 [Шаблон словаря данных](docs/DataDictionary_Template.xlsx)

## 🎯 Критерии оценивания

| Оценка | Требования |
|:------:|:-----------|
| **5 (отлично)** | ✅ Создана БД<br>✅ Таблицы заполнены тестовыми данными (≥3 записей)<br>✅ Создан словарь данных |
| **4 (хорошо)** | ✅ Создана БД<br>✅ Создан словарь данных<br>❌ Тестовые данные отсутствуют или менее 3 записей |
| **3 (удовлетворительно)** | ✅ Создана БД<br>❌ Словарь данных отсутствует<br>❌ Тестовые данные отсутствуют |
