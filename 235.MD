### 📝 Как пользоваться таблицей

- **🖥️ Компьютер** — локальные учётные данные для входа в систему (общие для всех).
- **🐙 GIT (Gogs)** — доступ к Git-серверу (`192.168.2.202:3000`). Логин и пароль — как в столбце `#` и в данных PostgreSQL.
- **🐘 PostgreSQL** — данные для подключения к базе данных (хост `192.168.2.202`, порт `5432`). Имя базы данных: `demoN` (где N — номер пользователя).
- **Email** — сгенерирован по шаблону `userN@mail.ru` для заполнения профиля в Gogs.

Если нужно добавить ещё студентов (у вас в списке 30 человек, а в SQL-командах — до 38), просто продолжите таблицу по тому же принципу.
